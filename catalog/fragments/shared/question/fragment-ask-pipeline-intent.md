# Fragment: Ask Pipeline Intent

Ask the user what they want the new pipeline to accomplish.

## Question
What should this pipeline do? Describe the goal or task you want to automate.

## Guidelines for User
- What is the main objective?
- Who will use this pipeline?
- What domain does it belong to?
- Are there specific inputs or outputs?

## Capture
- Pipeline goal
- Target domain
- Any constraints mentioned
